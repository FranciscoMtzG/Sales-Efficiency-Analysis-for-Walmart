# Sales Efficiency Analysis for Walmart

## 📊 Objective
Analyze Walmart’s sales efficiency to identify trends, top-performing categories, and improvement opportunities.

## 📁 Dataset
- Source: Walmart sales dataset
- Rows: 97,000
- Columns: 12
- Key variables: Date, Store, Category, Sales, Profit

## 🛠️ Tools Used
- Excel
- Pivot Tables
- Formula


## 🔍 Methodology
1. Data cleaning and preprocessing  
2. Exploratory and data analysis (EDA)  
3. Joinning data tables by Vlookup   
4. KPI calculation (sales per store, profit margin)
5. Created KPIs dashboards
6. Apply QA to ensure analysis quality.
7. Executive report using CFI method issuing business recommendations.

## 📈 Key Insights
- Grocery and basics category had the highest profit margin.  
- Grocery and basics category had higher percentage contribution to total sales..  
- All Stores Garden department with lower sales contributions. .

## 🧠 Conclusions
- Ensure product inventory for Grocery and basics department all stores.
- Reduce store space for Garden department.
